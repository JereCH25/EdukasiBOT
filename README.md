## Edukasi Bot 
Edukasi Bot Whatsapp

## Clone this project

```bash
> git clone https://github.com/JereGanzz/edukasibot-main
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd edukasibot-main
> bash install.sh
```

### Usage
```bash
> npm start
```

### Settings
```json
{
	"prefix": "#",
}
```
